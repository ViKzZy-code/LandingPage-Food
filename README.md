# 🍽️ Landing Page - Restaurante

Landing page moderna e responsiva para restaurante, desenvolvida utilizando **HTML, CSS e JavaScript**, com foco em uma apresentação visual atrativa e uma experiência de navegação simples e intuitiva.

## 🚀 Tecnologias utilizadas

* **HTML5** → Estruturação da página e organização dos conteúdos.
* **CSS3** → Estilização, responsividade, animações e criação do layout visual.
* **JavaScript** → Implementação de interações dinâmicas e melhorias na experiência do usuário.

## ✨ Funcionalidades

* 📱 Layout responsivo para diferentes dispositivos.
* 🍔 Apresentação do cardápio e pratos principais.
* 🖼️ Galeria de imagens.
* 🎨 Animações e efeitos visuais.
* 📌 Navegação entre seções da página.
* 📞 Área de contato.
* 🔗 Botões de ação para interação com o usuário.

## 📚 Aprendizados

Durante o desenvolvimento do projeto, foram aprimorados conhecimentos em:

* Criação de páginas web responsivas.
* Estruturação semântica utilizando HTML5.
* Desenvolvimento de layouts modernos com CSS3.
* Aplicação de animações e efeitos visuais.
* Manipulação de elementos da página com JavaScript.
* Organização e boas práticas no desenvolvimento Front-End.
